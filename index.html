What is ONESOURCE E-Invoicing?

ONESOURCE E-Invoicing is a centralized platform designed to streamline electronic invoicing management for multinational businesses, offering pre-built integrations connecting financial systems, including common enterprise resource planning (ERP) systems.

Global Coverage

Compliance with e-invoicing mandates in 80+ countries with support for all Continuous Transaction Control (CTC) models.

Format Conversion

Automatic format conversion between 50+ e-invoicing formats. Submit once, deliver to any recipient format.

Real-time Validation

Real-time validation and clearance with tax authorities. Pre-validate documents before submission.

Network Connectivity

Connect to 14 million businesses globally through the ONESOURCE Pagero network.

Key Concepts

Partner Integration

Your organization receives one set of universal credentials for all clients. Build the integration once, serve many clients without managing client-specific credentials.

Multi-Tenancy

Each client has one or more legal entities (companies), identified by a unique companyId. Users authenticate via OAuth 2.0 to grant your platform access.

Universal Format (PUF)

Submit documents in Pagero Universal Format (PUF) XML. ONESOURCE automatically converts to the recipient's required format (FatturaPA, CFDI, Factur-X, etc.).

Document Lifecycle

Documents flow through multiple statuses: submission → validation → clearance (CTC countries) → delivery → business acknowledgment. Track status via polling APIs.
